# Pilot Validation Docs Branch

This folder is a dedicated documentation branch for pilot repository validation.

Contents:
- `PILOT_REPO_REGISTRY.md` - pinned open-source repo set
- `PILOT_TEST_PROTOCOL.md` - run protocol and pass criteria
- `PILOT_RESULTS_TEMPLATE.md` - reporting template for run results
- `PERSONAL_MONITOR_QUICKSTART.md` - one-click monitor usage guide
- `PILOT_RESULTS_BATCH1_2026-03-17.md` - first executed batch (3 repos x 3 profiles)
- `PILOT_RESULTS_FULL_2026-03-17.md` - full 12x3 acceptance matrix (36 runs)
