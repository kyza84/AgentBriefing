# docs/archive

Архивная зона для закрытых/исторических материалов.

## Разделы
1. `specs/` - устаревшие или закрытые спецификации.
2. `cycle_plans/` - завершенные планы циклов и hot-fix документов.
3. `pilot_validation/reports/` - результаты пилотных прогонов.
4. `pilot_validation/protocol/` - протоколы и шаблоны валидации.
5. `tracker_history/` - снимки и история мастер-трекера.

## Правило
- В `docs/` (корень) остается только ядро + активные документы.
- Всё завершенное переносится сюда, чтобы не раздувать рабочий контур.
- Карта переноса от 2026-03-18: `docs/archive/DOCS_MIGRATION_MAP_2026-03-18.md`.
